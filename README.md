# 功能介绍

## 多边形截图
默认矩形截图，支持用户选择边数，最大值无穷，为圆形

## 键位

`ALT + Z`: start to capture
`Ctrl + Z`: undo
`Ctrl + Shift + Z / Ctrl + Y`: redo
`Ctrl + 1`: 切换选择模式
`Ctrl + 2`: cir
`Ctrl + 0`: back to default

`left click`: select a start point
`right click`: select a end point and close the shape
